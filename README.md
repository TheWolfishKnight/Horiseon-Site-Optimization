## About The Project


I had to take the existing Horiseon Site and optimize its code while leaving it clean and easy-to-understand for the next coder who tries to take it on.

Here's why:
The code was a mess and needed to be cleaned up.

I need to practice.

I needed to fix the nav links.

Goals Reached: Retagging and adding semantic tags, consolidated CSS code, commented CSS code, fixed nav links, fixed hero image.
